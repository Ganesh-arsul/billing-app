# Billing App

This is a React + Redux Toolkit + TypeScript application.

## Features

- Add products to cart
- Increase / decrease quantity
- Special offers:
  - Cheese: Buy 1 Get 1 Free
  - Soup + Bread: Bread at half price
  - Butter: 1/3 discount
- Subtotal, savings and final total calculation

## Tech Stack

- React
- Redux Toolkit
- TypeScript
- Tailwind CSS
